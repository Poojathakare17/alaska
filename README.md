#[Alaska](https://github.com/maichong/alaska)

Alaska is componentized and pluggable web framework for Node.js based on Koa , Mongoose and React

```

npm install -g alaska-cli

mkdir demo

cd demo

alaska init

alaska build

node index.js

```
